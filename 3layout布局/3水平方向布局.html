元素在其母元素中水平方向的位置有以下几个属性决定
  • margin-left
  • border-left
  • padding-left
  • width
  • padding-right
  • border-right
  • margin-right
    一个元素在其母元素中水平布局必须满足以下等式
    margin-left + border-left + padding-left + width + padding-right + border-right + margin-right =其母元素内容区的宽度
  • 如果七值中没有为 auto 的情况，浏览器会自动调整margin-right值
  • 七值中，width（默认值为auto），margin-left，margin-right可设为 auto，有auto会自动设置auto使等式成立
  • 若设置width和一个margin为auto，则width为最大，那个margin为0
  • 若三个值都为auto，则width最大，margin都为0
  • 若设置margin都为auto，width固定，则两个margin值相同（经常使用此来水平居中）
